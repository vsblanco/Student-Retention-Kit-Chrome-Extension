# Privacy Policy for Student Retention Kit

**Last Updated:** 12-19-2025

## Overview
Student Retention Kit is a Chrome extension developed exclusively for Northbridge University retention staff to monitor at-risk students and conduct outreach calls through the university's Canvas LMS and Five9 call center systems.

## Intended Use
This extension is designed **exclusively for authorized Northbridge University retention staff** and is not intended for:
- General public use
- Use by other educational institutions
- Use by students or unauthorized personnel
- Commercial purposes

## Data Collection

### What Data We Collect
- **Student Information:** Names, student ID numbers, phone numbers, Canvas user IDs of Northbridge University students
- **Academic Data:** Grades, missing assignments, course enrollment, and submission history from Northbridge University Canvas courses
- **Canvas Activity:** Page visits and assignment submissions within Northbridge University Canvas LMS (northbridge.instructure.com)
- **Call Records:** Call dispositions and notes for student outreach (stored locally)

### How We Collect Data
- Data is imported from Northbridge University student information systems (CSV/Excel files)
- Academic data is fetched via Northbridge University Canvas LMS API using your authenticated Northbridge credentials
- Canvas activity is monitored only on the authorized Northbridge University Canvas domain (northbridge.instructure.com)

## Data Storage
- **All student data is stored locally** in your browser using Chrome's storage API
- **No data is transmitted** to external servers except:
  - Authenticated API calls to Northbridge University Canvas LMS (northbridge.instructure.com)
  - Authenticated API calls to Northbridge University Five9 call center (app-atl.five9.com)
  - Optional exports to Microsoft Office/SharePoint (user-initiated, for internal Northbridge reporting)

## Data Usage
- Data is used **solely** for Northbridge University student retention tracking and outreach management
- We **do not sell** or transfer student data to third parties
- We **do not use** student data for purposes unrelated to Northbridge University student retention activities
- We **do not use** student data for creditworthiness or lending purposes

## Data Access
- Student data is only accessible to **authorized Northbridge University retention staff** who are authenticated in Canvas and Five9
- Extension operates within the **same permissions** granted to you by Northbridge University's Canvas and Five9 systems
- No unauthorized access to Northbridge University student records

## Data Security
- Student data is protected by Chrome's built-in security features
- Authentication is handled by Northbridge University's Canvas and Five9 authentication systems
- No passwords or credentials are stored by the extension
- All data access requires valid Northbridge University credentials

## Data Retention
- Student data remains in local storage until:
  - You manually clear it using the extension
  - You uninstall the extension
  - You clear your browser data
- Data is subject to Northbridge University's data retention policies

## Third-Party Services
The extension integrates exclusively with Northbridge University authorized systems:
- **Northbridge University Canvas LMS** (northbridge.instructure.com) - For student academic data
- **Northbridge University Five9** (app-atl.five9.com) - For call center functionality
- **Microsoft Office/SharePoint** (optional) - For internal Northbridge reporting and data export

All integrations require your existing authenticated Northbridge University sessions.

## FERPA Compliance
This extension is designed to comply with the Family Educational Rights and Privacy Act (FERPA) and Northbridge University's policies regarding student educational records:
- Access is limited to authorized university officials with legitimate educational interest
- Student data is used solely for retention and student success purposes
- No student data is disclosed to unauthorized parties

## User Rights
As an authorized Northbridge University staff member, you have the right to:
- Access stored student data via the extension interface
- Delete stored data by clearing storage or uninstalling
- Export data for internal Northbridge reporting purposes

## Children's Privacy
This extension is designed exclusively for Northbridge University staff members and is not intended for use by students or children under 13.

## Institutional Authorization
Use of this extension constitutes acceptance of:
- Northbridge University's acceptable use policies
- Northbridge University's data privacy and security requirements
- Your responsibilities as an authorized university official handling student records

## Changes to Privacy Policy
We may update this privacy policy to reflect changes in Northbridge University requirements or functionality. The "Last Updated" date will reflect any changes.

## Compliance
This extension complies with:
- Chrome Web Store Developer Program Policies
- FERPA (Family Educational Rights and Privacy Act) guidelines for educational records
- Northbridge University institutional data privacy and security requirements
- Northbridge University acceptable use policies
