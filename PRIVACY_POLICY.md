# Privacy Policy for Student Retention Kit

**Last Updated:** [Current Date]

## Overview
Student Retention Kit is a Chrome extension designed for educational institutions to help retention staff monitor at-risk students and conduct outreach calls.

## Data Collection

### What Data We Collect
- **Student Information:** Names, student ID numbers, phone numbers, Canvas user IDs
- **Academic Data:** Grades, missing assignments, course enrollment, submission history
- **Canvas Activity:** Page visits and assignment submissions within Canvas LMS
- **Call Records:** Call dispositions and notes (stored locally)

### How We Collect Data
- Data is imported from institutional student information systems (CSV/Excel files)
- Academic data is fetched via Canvas LMS API using your authenticated session
- Canvas activity is monitored only on authorized Canvas domains

## Data Storage
- **All data is stored locally** in your browser using Chrome's storage API
- **No data is transmitted** to external servers except:
  - Authenticated API calls to Canvas LMS (nuc.instructure.com)
  - Authenticated API calls to Five9 call center (app-atl.five9.com)
  - Optional exports to Microsoft Office/SharePoint (user-initiated)

## Data Usage
- Data is used **solely** for student retention tracking and outreach management
- We **do not sell** or transfer user data to third parties
- We **do not use** data for purposes unrelated to student retention
- We **do not use** data for creditworthiness or lending purposes

## Data Access
- Data is only accessible to the **authenticated user**
- Extension operates within the **same permissions** you already have in Canvas and Five9
- No unauthorized access to student records

## Data Security
- Data is protected by Chrome's built-in security features
- Authentication is handled by your institution's Canvas and Five9 systems
- No passwords or credentials are stored by the extension

## Data Retention
- Data remains in local storage until:
  - You manually clear it using the extension
  - You uninstall the extension
  - You clear your browser data

## Third-Party Services
The extension integrates with:
- **Canvas LMS** (nuc.instructure.com) - For student academic data
- **Five9** (app-atl.five9.com) - For call center functionality
- **Microsoft Office/SharePoint** (optional) - For data export

All integrations require your existing authenticated sessions.

## User Rights
You have the right to:
- Access your stored data via the extension interface
- Delete your data by clearing storage or uninstalling
- Export your data using the download feature

## Children's Privacy
This extension is designed for educational institution staff only and is not intended for use by children under 13.
